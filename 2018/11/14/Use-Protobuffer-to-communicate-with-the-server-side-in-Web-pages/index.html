在 Web 页面中使用 Protobuffer 与服务端通信

2018-11-14   ProtoBuffer 

protobuffer（下面的内容中简称pb）是 google 的一种数据交换的格式，它独立于语言，独立于平台。由于它是一种二进制的格式，比使用 xml 或者 json 进行数据交换快许多。作为一种效率和兼容性都很优秀的二进制数据传输格式，可以用于诸如网络传输、配置文件、数据存储等诸多领域。
