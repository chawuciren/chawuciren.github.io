开启 Sublime Text 的 VIM 模式

2018-11-14   VIM  Sublime Text 

经常在切换到 Sublime 中编辑一些文本时，不自觉地敲了 VIM 的命令。其实 Sublime 也支持使用 VIM 模式来编辑文件，但是需要在配置中自己开启。


1.在菜单栏中： Preferences -> Setting - User ，即可打开配置文件进行编辑，将 ignored_packages 项的[]里面内容清空：


2.关闭 Sublime 并重新启动：

这时 VIM 模式已经被启用，在编辑前按Esc键退出编辑模式，就进入 VIM 模式了。
